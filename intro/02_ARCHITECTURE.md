## дҫқиіҙж–№еҗ‘

дҫқиіҙж–№еҗ‘жҳҜеҡҙж јзҡ„пјҢе®ҡзҫ©еңЁ `TOOLING_USAGE_RULE.md`пјҡ

```
rules в”Җв”ҖвҶ’ (иў«и®ҖеҸ–) в”Җв”ҖвҶ’ жүҖжңүжЁЎзө„

devtools в”Җв”ҖвҶ’ (еҸҜеј•з”Ё) в”Җв”ҖвҶ’ core_lib
skills  в”Җв”ҖвҶ’ (еҸҜеј•з”Ё) в”Җв”ҖвҶ’ core_lib
agents  в”Җв”ҖвҶ’ (еҸҜеј•з”Ё) в”Җв”ҖвҶ’ core_lib + skills

core_lib в”Җв”Җ/Xв”Җв”ҖвҶ’ devtools   пјҲзҰҒжӯўеҸҚеҗ‘дҫқиіҙпјү
core_lib в”Җв”Җ/Xв”Җв”ҖвҶ’ agents     пјҲзҰҒжӯўеҸҚеҗ‘дҫқиіҙпјү
core_lib в”Җв”Җ/Xв”Җв”ҖвҶ’ skills     пјҲзҰҒжӯўеҸҚеҗ‘дҫқиіҙпјү
```

ж ёеҝғеҺҹеүҮпјҡ**`core_lib` жҳҜ runtime дҫқиіҙпјҢ`devtools` жҳҜ dev дҫқиіҙгҖӮ** `core_lib` дёҚиғҪеј•з”Ёд»»дҪ•дёҠеұӨжЁЎзө„гҖӮ
